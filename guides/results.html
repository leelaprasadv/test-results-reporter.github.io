# Results

This library uses test-results-parser (opens new window) to read test results from different test frameworks.

# Supports

Test results reporter type.

âœ… TestNG

# Arguments

# > type (string)

Name of the test runner.

# > files (string[])

Path to the test result files.

LIMITATIONS

  • Currently it supports only one test results report file.
  • No support for wildcard paths.
