# Hyperlinks

Hyperlinks extension will attach links to your teams or slack reports.

Defaults

DEFAULTS
  • hook: end
  • condition: passOrFail

# Inputs

# > links (object[])

Array of link objects

# > link.text (string)

Text to display on the message.

# > link.url (string)

Url to link to the given text in the message.
